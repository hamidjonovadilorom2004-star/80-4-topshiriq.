# 5-Topshiriq: Frontend interfeysni ishlab chiqish

## Bajarilgan ishlar
Frontend `frontend/` papkasida yozildi.

## Asosiy fayllar

- `frontend/index.html` — UI struktura
- `frontend/styles.css` — dizayn va responsiv holat
- `frontend/app.js` — interaction logikasi
- `frontend/abi.js` — kontrakt ABI
- `frontend/config.js` — chain va contract address konfiguratsiyasi

## UI bo'limlari

1. Wallet ulash (MetaMask)
2. Statistika paneli (mint fee, NFT soni, proposal soni)
3. NFT mint formasi
4. NFT update/burn formasi
5. Proposal yaratish formasi
6. Proposal update/delete formasi
7. Proposal list + vote/execute tugmalari
