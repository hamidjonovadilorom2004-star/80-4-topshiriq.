# 8-Topshiriq: Dizayn va UX yaxshilanishlari

## Vizual qarorlar

- Maqsad: oddiy, toza va zamonaviy Web3 panel.
- Ranglar: yashil/iliq accent palitra (`--brand`, `--accent`) bilan.
- Tipografiya: `Manrope` font.
- Fon: radial gradient + blur glow elementlar.

## UX yaxshilanishlari

1. Responsive layout:
- Desktopda 2 ustun, mobil qurilmada 1 ustun.

2. Tushunarli feedback:
- Har tranzaksiya uchun toast xabar.
- Error xabarlari ham chiqariladi.

3. Oqimni soddalashtirish:
- Wallet ulanishdan keyin mint `to` fieldiga address auto qo'yiladi.
- Refresh tugmasi bilan tezkor holat yangilash.

4. Harakatlar:
- Tugmalar hover animatsiyasi.
- Card lar page load paytida yumshoq animatsiya.
